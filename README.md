# RubyでFizzBuzzに挑戦


## 【要件】
- 1～任意の数までの数字を順番に表示する
- ただし、3の倍数のときはFizz、5の倍数のときはBuzz、3と5の公倍数のときはFizzBuzzと表示する。
- 任意の数はプログラム実行時に引数で渡せるようにする

## 【実行例】

```
$ ruby fizzbuzz.rb 30
1

"Fizz!"
4
"Buzz!"
"Fizz!"

"Buzz!"

"FizzBuzz!"
```
